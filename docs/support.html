Support

帮助与支持

关于小组件添加、每日更新、推荐切换和豆瓣跳转的常见问题。

如何添加小组件？

长按 iPhone 主屏幕空白处，选择“编辑”与“添加小组件”，搜索“电影日历”或“CineCal”，然后选择小号或中号尺寸。

小组件为什么没有立即更新？

打开 CineCal 主应用并点击“换一部”，或点击小组件上的切换按钮。iOS 可能需要几秒钟刷新小组件。请同时确认设备已联网。

每日推荐何时刷新？

推荐日期跟随手机当前时区。跨过本地午夜后，小组件会在系统允许的刷新时机更新。

仍然需要帮助？

请在 GitHub Issues 提交问题，并附上 iOS 版本、设备型号和问题截图。请勿公开发送 Apple ID、密码或其他敏感信息。
